# 03. 診断コメントテンプレート一覧

## 基本方針

- 短い
- 断定しすぎない
- 実行に結びつく

## 出力フォーマット

各クリップで返す項目:

- `summary`
- `strengths[1]`
- `weaknesses[1..2]`
- `labels[]`
- `next_focus`

## ラベル別テンプレート

### `slow_initial_adjustment`

summary:

- 敵を捉えるまでに少し時間がかかっている
- 初弾までの合わせにやや迷いが見える

strengths:

- 射撃開始後の維持は大きく崩れていない
- 視認後に完全には置いていかれていない

weaknesses:

- 最初の合わせで余分な修正が入っている
- 視認直後に大きく振りすぎている

next_focus:

- 視認した瞬間に大きく振らず、最短で中央へ寄せる

### `overflick`

summary:

- 合わせた後に少し行き過ぎる傾向がある
- 初動で目標を通り越しやすい

strengths:

- 反応速度自体は悪くない

weaknesses:

- 止める動作よりも振る動作が強い
- 細かい修正回数が増えている

next_focus:

- 一度で大きく合わせず、小さく止める意識を持つ

### `tracking_delay`

summary:

- 中盤以降の追いエイムで少し遅れが出ている
- 敵の横移動に対して後追いになりやすい

strengths:

- 開始直後の合わせは比較的速い

weaknesses:

- 敵の移動に対して視点が遅れて追従している
- 当たり始めた後に維持しきれていない

next_focus:

- 敵の少し先をなぞる感覚で追従する

### `recoil_breakdown`

summary:

- 射撃の後半で安定が崩れやすい
- リコイル制御の維持にばらつきがある

strengths:

- 撃ち始めの数発はまとまりやすい

weaknesses:

- 継続射撃で視点が上下左右に散りやすい
- 維持より修正が先に増えている

next_focus:

- 撃ち始めよりも後半の維持を優先して意識する

### `close_range_instability`

summary:

- 近距離で視点が暴れやすい
- 敵が近い場面で追い方が不安定になっている

strengths:

- 視認そのものは早い

weaknesses:

- 左右移動に反応しすぎて振られている
- 近距離で細かい補正が増えている

next_focus:

- 胸元付近を安定させ、追いすぎない

### `ads_judgment_issue`

summary:

- 近距離で ADS 判断が噛み合っていない可能性がある
- 距離に対して構え判断が少し重い

strengths:

- 距離感の大きな破綻はない

weaknesses:

- 近距離で ADS に入ることで追従が窮屈になっている
- 腰撃ちで十分な場面でも切り替えが遅い

next_focus:

- 近距離は腰撃ち優先で安定を取る

## confidence 別の言い回し

`confidence >= 0.75`

- はっきり傾向が出ている
- 比較的一貫して見られる

`0.50 <= confidence < 0.75`

- ややその傾向がある
- この場面ではそう見える

`confidence < 0.50`

- その可能性がある
- 断定はしにくいが気になる
